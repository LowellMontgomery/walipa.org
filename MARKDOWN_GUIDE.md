# Walipa Community Markdown Guide

To help build our documentation, use these symbols in your text:

### **Headers**
`# Main Title`
`## Section Title`
`### Subsection`

### **Emphasis**
`**Bold Text**` for important notes.
`*Italic Text*` for botanical names like *Chamaebatia foliolosa*.

### **Lists & Links**
`* Item 1`
`[Link Text](https://walipa.org)`

### **Quotes**
`> "I have a dream..."`

---
(Three dashes create a horizontal separator)
