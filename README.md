# How to Format Walipa Documents

To create a heading, use:
`# Your Title`

To create bold text, use:
`**your text**`

To create a list, use:
`* Item 1`
`* Item 2`

To insert a link:
`[Link Name](URL)`

To show code or specific commands:
\`your command here\`
